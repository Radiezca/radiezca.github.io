Projects

   Genetic Zelda Levels - Java application that creates randomized levels for a Legend Of Zelda’s style using a Feasible-Infeasible Two Populations Genetic Algorithm, generates a JSON file that after can be seen in a Unity simple level renderer.
   Parseros - Java Web application that translates a PHP project into a Node.js project running through the folders of the project, the translation of the PHP’s grammar is possible using ANTLR v4.
   UN-App - Web application for the students of the Universidad Nacional de Colombia, where they can share their opinions of the classes and teachers over their time in the university, the students can review teachers and subjects, the objective of the application is to improve the subject contents, the teacher methodology and in general the learning process in the university.
